Set or change counting channel. Current Channel:{{counting_channel}}

Counting is a simple game that can increase activity in a server. One user counts and another user says the next number however there are a few rules.
A user cannot count twice in a row.
If a user says a number that is not the next number in the counting sequence it resets the counting sequence.
Numbers can be in other forms such as math equations, binary, bit shift, etc.
Each user and server gets there own stats this will show accuracy, highest number, and build a leaderboard.
(To disable counting enter None)
